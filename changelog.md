# Technocratica Changelogs

## v3.1.0

[Click here for a more detailed changelog](https://github.com/Wxrlds/Technocratica/compare/release/v3.0.09...release/v3.1.0)

This is a somewhat big update that replaces a lot of abandoned mods with maintained versions of them.
Most things should run fine, but expect breakage with ExtraCells2 fluid and gas cells and SimplyJetpacks items changing into other items.
New chunks might get cut off due to changes in terrain generation. Your existing chunks will not be impacted by this!
Definitely make a backup before you update!

- Disabled Update checkers for Actually Additions; OpenComputers; LunatriusCore
- Disabled first chat join message for Hats
- Fixed broken recipe for Ender IO Endergy Exchanger Core (Tier 2)
- Added Klein Stars to the list of allowed items in the Stone Transmutation Table, since they work anyway
- Added random saplings to the Bonsai Pot quest reward as well as a rubber tree
- Disabled all The One Probe items as they are not required to check block info
- Added Biosphere Portal frames to the quest reward for the mining dimension
- The wireless ME Terminals from WCT Lib now require the new wireless Terminals from AE2
- Removed recipe for Extra Cells Wireless Fluid Terminal
- Adjusted recipes from Extra Cells to AE Additions
- Stone of Inertia Null now does make you immune to fall damage
- The main menu now looks a lot better at different display resolutions
- Reformatted and re-ordered all recipe scripts
- Reset the Biomes o Plenty config to new default values
- Enderman no longer pick up blocks
- Disable Enderman anti cheese
- Reduced log spam from Biomes o Plenty
